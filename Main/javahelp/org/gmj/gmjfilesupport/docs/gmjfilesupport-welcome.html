Welcome to G-java

Thank you for downloading G-java. The documentation for G-java is split into 3 main subtopics:

Using G-java
This section describes how to use G-java. It describes the main idea and takes the user through a few examples. No programming required, all games are made using the "drag and drop" action librarys that come with G-java. This section goes on to describe each action and should be used as a reference when using the default action librarys.

G-Java Language API
This section introduces the G-Java language API, which is used to extend the functionality of g-java beyond the action librarys, and even allow the user to create there own actions. With the G-java API users can have full control over their game using easy to use functions in a proper programming language - The Java Language. This section should be used with G-java language tutorials to teach non-programmers quickly and easilly, so they can concentrate on their game/software.

Distributing Software
This section will describe how to distribute your software for a wide range of platforms.
