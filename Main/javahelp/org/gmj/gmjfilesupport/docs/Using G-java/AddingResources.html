Adding resources

To add images, sounds, fonts or any other resources all you have to do is drag the files and drop them onto the G-java treview into the correct folder, for example images MUST be places in the images folder!

It can also be noted that you can drag files out of the G-java treeview and put them into folders if you ever want to move files around this can be very useful.

If you want to add standard G-java files then use the toolbar, or right click a folder and select new -> other -> then click G-java files -> from here you can select:

Room
Actor
Action

You can also use the toolbar to add images and sounds by browsing for the files instead of dropping them in the tree.
